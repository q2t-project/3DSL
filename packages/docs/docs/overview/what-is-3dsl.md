# 3DSL とは

3DSL は、文章だけでは取りこぼしやすい「関係」「距離」「俯瞰」を、**図（3D）として扱えるようにする**ための表現・閲覧環境（Viewer）と、その周辺運用（SSOT/同期）を含む試み。

ざっくり言うと：

- **3DSS（3D Structured Scene）**としてデータ化して保存できる
- Viewer で「触って」「戻って」「深掘り」できる（回転・ズーム・視点移動など）
- 文章（Docs / FAQ）と図（Library / Viewer）を行き来できる

入門は次の順が分かりやすい：

1. **Library** のカードから気になるモデルを開く
2. Viewer で関係を触って掴む
3. Docs / FAQ で背景・仕様・運用を読む
