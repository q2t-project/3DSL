## 1. 事前許諾不要でできること

### 1.1 リンク
本サイトのページへのリンク（URL 共有、SNS 投稿、紹介記事等）は自由です。

### 1.2 引用（スクリーンショット等）
報道・批評・研究等の目的で、法令で認められる「引用」の範囲内に限り、スクリーンショット等の利用が可能です。
引用部分の明確な区別、主従関係、必要最小限、出所の明示など、引用の要件を満たしてください。

※引用の可否は最終的に法令・判例実務に従います。本条件は、法令に基づく利用を制限するものではありません。

### 1.3 閲覧に伴う一時的利用
ブラウザ等の通常動作に伴い不可避に生じる一時的なキャッシュ等は、閲覧の範囲で許容されます。

## 2. 事前許諾なく禁止すること

権利者の事前の書面等による許諾なく、以下の行為を禁止します（例示であり、これらに限りません）。

### 2.1 データの複製・再配布・転載

- データファイルの配布、アーカイブ化、共有ドライブへの保存・共有

### 2.2 第三者環境への取り込み・再提供（Viewer 等を含む）
- 本コンテンツを別サイト・別アプリ・別ビューア等に取り込み、表示・操作可能な形で提供する行為

- API 化、データベース化、データセット化、学習用コーパス化

### 2.3 改変・派生物の作成および配布
- 変形、翻案、加工、部分抽出した再構成等（ただし法令上許される範囲を除く）

### 2.4 自動収集・回避
- スクレイピング、クローリング、機械的収集、解析目的の大量アクセス
- アクセス制御の回避

### 2.5 権利表示の改変
- 著作権表示、クレジット、注意書き等の削除・改変・秘匿
