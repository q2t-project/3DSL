このサイトは、三次元（3D）を使って **概念・情報・構造** を整理して、文章だけやと掴みにくい「関係性」「距離感」「俯瞰」を見える形にするための実験的なプラットフォームや。

- 学習・研究・思考整理を主目的にしとる
- 自動生成コンテンツを大量にばら撒く用途は想定してへん
- 文章（Docs/FAQ）と図（Library/Viewer）を行ったり来たりできるように作ってる

まずは Library から気になったカードを開いて、Viewer で触ってみてな。
