# Repo Navigation (地図・索引・配線)

このフォルダは、3DSL リポジトリを「迷わず触る」ための **地図（MAP）/索引（INDEX）/配線図（WIRING）** を集約する。

- **システム仕様SSOTそのものではない。** ただし SSOT の“場所”と“読み方”を明確化する。
- 生成物・ミラー（例: `apps/site/public/**`, `dist/public/vendor`）は編集せず、必ず SSOT 側を直す。

## 入口

- [MAP.md](./MAP.md): 全体像（フォルダ分類・層・主なデータフロー）
- [INDEX.md](./INDEX.md): 用語/逆引き索引（検索で引ける辞書）
- [NAV_POLICY.md](./NAV_POLICY.md): 運用規範（どこに何を書くか、更新ルール）
