# Viewer
---

## 図を「触って」理解するための閲覧環境です。

3DSL Viewer は、3DSS（図データ）を**読み取り専用**で表示するための環境です。  
図は「眺める→気になった箇所へ寄る→引いて全体を見る」を往復するほど、理解が早くなります。

**使い方**

- まず `/library` から図を選び、Viewer を開きます  
- 直接 URL を指定して開くこともできます（共有リンク向け）

**基本操作（目安）**

- 回転：ドラッグ  
- ズーム：ホイール／ピンチ  
- 移動：右ドラッグ／二本指ドラッグ（環境によって異なる場合があります）

詳細は、Viewer 内のヘルプ表示（ある場合）と Docs を参照してください。
