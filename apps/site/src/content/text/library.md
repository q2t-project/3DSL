# Library
---

## サンプルは完成形ではなく、思考の足場です。

Library には、さまざまなテーマの 3DSL サンプルを収録します。これらは正解集ではありません。世界への違和感を手がかりに、概念を配置し、関係を結び、全体と局面を往復できる構造物として組み上げる試みです。

役に立つ構造は、使われることで鍛えられ、改訂され、共有されていきます。内容が真実であるかどうかよりも先に、理解や思考の足場として機能するかどうかが重要な場面があります。

### 著作権について

本ページに掲載されている図およびスナップショットは、各原著作物を正当に引用し、構造的理解を目的として再構成したものです。著作権はそれぞれの権利者に帰属します。操作による探索は、各コンテンツに付随するリンク先 Viewer にて行ってください。
