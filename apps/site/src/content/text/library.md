# Library
---

## サンプルは完成形ではなく、思考の足場です。

Library には、3DSS（図データ）のサンプルを掲載します。作品として完成した「最終図」よりも、考えを整理し、関係を試し、改善していくための素材を多く置きます。

**掲載の方針**

- 学習・研究・理解促進を目的とした公開コンテンツを中心にします  
- 「新着」と「人気」の切り替えができるようにしつつ、ページ自体は軽量に保ちます  
- 広告は Library には載せず、コンテンツ閲覧ページで表示します（読みやすさ優先）

**引用と著作権**

Library に掲載される図は、原著作物を正当に引用し、その関係構造を可視化したものです。図は原著作物の代替閲覧を目的とするものではありません。詳細な参照は、出典元またはリンク先（Viewer）で行ってください。著作権はそれぞれの権利者に帰属します。
