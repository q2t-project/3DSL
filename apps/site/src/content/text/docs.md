## なぜ図なのか。なぜスケールなのか。

Docs では、3DSL の考え方と設計思想、データ形式、表現上の約束事をまとめます。

図は強力な表現である一方、自由度が高すぎて、場面ごとに恣意的なルールが乱立しやすいという問題があります。3DSL は、図を「関係の言語」として扱うために、最低限の枠組みを用意します。

### 発散と収束

理解や探索は、視野を広げる局面（発散）と、追い込み形を固める局面（収束）を往復します。マルチスケール UI は、この往復を道具として支えます。

### 引用と表現

3DSL Library に掲載される図は、研究・教育・理解促進を目的として、原著作物を正当に引用し、その関係構造を可視化したものです。図は原著作物の代替閲覧を目的とするものではなく、内容の探索や詳細な参照は出典元またはリンク先 Viewer に委ねられています。各著作物の著作権は、それぞれの権利者に帰属します。
