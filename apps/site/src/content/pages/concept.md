# Concept

言葉は輪郭を描き、関係は図として残る。  
このページは、その入口です。  
文章は最小限に留め、本体は 3DSS（図）として提示します。必要なら Viewer で回転・拡大し、俯瞰と局所を往還しながら読んでください。

## 1. 概要
3DSL は、概念（ラベル）と関係（リンク）を 3D 空間に配置して表現するための枠組みです。  
3DSS はその表現を機械可読にする JSON 形式であり、Viewer は 3DSS を読み込んで表示します。  
本サイトは Viewer、3DSS の公開情報、サンプル（Library）、および利用ドキュメント（Docs）を提供します。

## 2. 提供するもの
- **Viewer**：3DSS（.3dss.json）を読み込んで 3D 表示する Web アプリ
- **3DSS**：3DSL 表現を JSON として記述する形式（スキーマに基づく検証が可能）
- **Library**：公開サンプル（3DSS）の一覧
- **Docs**：使い方、仕様概要、検証、互換性、FAQ

## 3. できること
- 3DSS を表示し、概念と関係（ノード/リンク）の構造を閲覧する
- 視点操作により、俯瞰と局所（拡大/回転/移動）を往復しながら構造を確認する
- サンプル（Library）を参照して、自作 3DSS の作成・確認に利用する
- スキーマに基づく検証手順（Docs）を参照して、形式の整合性を確認する

## 4. できないこと（非目的）
- アカウント登録・会員機能の提供
- ユーザー投稿コンテンツのホスティング（投稿・コメント等）
- 個人情報や機密情報の保存を前提とした機能提供
- 医療・法律・投資など専門助言の提供
- サンプルや表示結果の正確性・適合性・完全性の保証

## 5. 想定読者
- 3DSS を閲覧したい人
- 3DSS を作成し、Viewer で動作確認したい人
- 3DSL の表現上の考え方を把握したい人

## 6. コンテンツの出所と参照
- 本サイトで公開している文書・サンプルは、運営側が作成・管理して公開します
- 外部参照（外部 URL や外部データ参照）を含む場合があります。取得先の安全性は利用者が判断してください

## 7. 注意事項（保証しない範囲）
- サンプルは動作例であり、内容の正確性・適合性を保証しません
- Viewer の表示結果はブラウザ環境やデータ規模により差異が出る場合があります
- 外部リンク/外部データ参照に伴う結果は利用者の責任で判断してください

## 8. バージョンと互換性
- 3DSS はスキーマ（Schema）とバージョンに従います
- スキーマ非適合、または互換性のないバージョンでは読み込みに失敗する場合があります  
  詳細は Docs の「Versioning」を参照してください（/docs/versioning など）
